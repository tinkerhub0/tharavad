Community Linux Server ‚ÄĒ TinkerHub Kerala
tharavad.xyz

What is this?

A shared Linux server for TinkerHub Kerala members.
Inspired by Hack Club Nest.
SSH in, build things, host projects, learn.

How to get access

  1. Sign up: https://auth.tharavad.xyz/if/flow/tinkerhub-enrollment/
  2. Wait for admin approval
  3. Add your SSH key: https://auth.tharavad.xyz/if/flow/add-ssh-key/
  4. SSH in: 
    ssh -p 2222 username@tharavad.xyz

What you get

  • Isolated bash shell
  • 5GB disk quota
  • CPU and memory resource limits
  • Run servers on your own ports
  • Common dev tools pre-installed (python, node, go, rust, git, ...)

SSH quick reference

# Generate SSH key (if you don't have one)
ssh-keygen -t ed25519 -C "your@email.com"
cat ~/.ssh/id_ed25519.pub
# paste at https://auth.tharavad.xyz/if/flow/add-ssh-key/

# Connect
ssh -p 2222 username@tharavad.xyz

# ~/.ssh/config shortcut
Host tharavad
    HostName tharavad.xyz
    Port 2222
    User username
# then just: ssh tharavad
