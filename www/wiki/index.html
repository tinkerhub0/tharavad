quick start
01
Request access
Sign up at the enrollment link. An admin approves your account before you can SSH in.
Full guide →
02
Add your SSH key
Generate an ed25519 keypair and paste your public key into your account.
Full guide →
03
Connect
SSH in on port 2222. Optionally add a shortcut to your SSH config.
ssh -p 2222 username@tharavad.xyz

Full guide →
04
Host something
Reserve a port, start your server inside tmux, and expose it at a public subdomain.
tinker port          # get a port number
tinker caddy myapp 45320  # → myapp.tharavad.xyz
