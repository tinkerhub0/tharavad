Getting started

01 — Request access

Sign up for an account. An admin will approve it shortly.

https://auth.tharavad.xyz/if/flow/tinkerhub-enrollment/

02 — Add your SSH key

Generate a key if you don't have one, then add it to your account.

# generate a key
ssh-keygen -t ed25519 -C "you@email.com"

# copy your public key
cat ~/.ssh/id_ed25519.pub

Paste it at: auth.tharavad.xyz/if/flow/add-ssh-key/

03 — SSH in

ssh -p 2222 username@tharavad.xyz

Optional — add to ~/.ssh/config for a shorter command:

Host tharavad
    HostName tharavad.xyz
    Port 2222
    User username
# then just
ssh tharavad

Hosting a project

Use tinker to get a port and expose your service publicly.

# reserve a port
tinker port
→ 45320

# start your server (use tmux so it keeps running)
tmux new -s myproject
python3 -m http.server 45320

# expose it at yourname.tharavad.xyz
tinker caddy yourname 45320

# or a custom subdomain
tinker caddy blog 45320
tinker caddy api 45321

tinker commands

tinker port                      reserve a port
tinker port list                 your reserved ports
tinker port release <port>       free a port

tinker caddy <subdomain> <port>  go live at subdomain.tharavad.xyz
tinker caddy list                your active subdomains
tinker caddy remove <subdomain>  take it offline

What you get

  • Isolated bash shell

  • Pre-installed: python, node, go, rust, git, vim, tmux, curl...
  • Public subdomains via tinker caddy
  • Ports auto-released after 7 days of inactivity
